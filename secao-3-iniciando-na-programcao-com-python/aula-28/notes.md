# Precedência entre os operadores aritméticos

| Precedência |                    Símbolos                    |
| :---------: | :--------------------------------------------: |
|     1º      |                      `()`                      |
|     2º      |                      `**`                      |
|     3º      |              `*`, `/`, `//`, `%`               |
|     4º      |                    `+`, `-`                    |
|     5º      | `==`, `!=`, `>`, `<`, `>=`, `<=` (relacionais) |
|     6º      |                 `not` (lógico)                 |
|     7º      |                 `and` (lógico)                 |
|     8º      |                 `or` (lógico)                  |
|     9º      |                `=` (atribuição)                |
