# Valores Truthy e Falsy, Tipos Mutáveis e Imutáveis

Os valores considerados _falsy_ em Python são:

- Lista vazia: `[]`

- Dicionário vazio: `{}`

- Conjunto vazio: `set()`

- Tupla vazia: `tuple()`

- String vazia: `""`

- Inteiro 0: `0`

- Flutuante 0.0: `0.0`

- Nada: `None`

- Falso: `False`

- Intervalo vazio: `range(0)`
